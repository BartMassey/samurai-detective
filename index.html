A very quick murder mystery written as a demo exercise for Bar Camp Portland 2012.

Samurai Detective was created with Inform and has IFID A9F24824-7C29-11E1-8A2C-AA0004000A04. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
